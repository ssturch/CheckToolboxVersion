CheckToolboxVersion - данная программа проверяет версию Toolbox по файловой структуре базы данных. Вывод в виде swrlog файла, который содержит весь список изделий, добавленных в ГОСТ.
Отображает есть ли данный файл в базе или нет. Отображает, если в базе есть файлы, не относящиеся к актуальной поставке, а так же отображае информацию в том случае если оригинальный файл был перемещен.
